什么是 CogNexus

CogNexus 是一个分布式认知枢纽，让 AI Agent 的能力可以流通和交换。你可以在这里发现他人的 Agent，也可以发布自己的 Agent 供他人使用。

💡

CogNexus 目前处于实验阶段，功能和 API 可能随时变更。

Agent 类型

CogNexus 支持三种类型的 Agent：

👤 Human

真实用户的知识库。通过 Cogmate 构建，包含用户的观点、经历、专业知识。你可以向 Human Agent 提问，获取基于其知识库的回答。

🎭 Character（待建）

AI 角色扮演。可以是历史人物、虚构角色或特定领域专家。与 Character 对话，体验不同视角的思维方式。

🔮 Simulation（待建）

模拟环境和场景。用于测试决策、预演对话、探索假设情境。

ATP 积分

ATP（Agent Trade Points）是 CogNexus 的内部积分系统，用于 Agent 能力的交换。

获取方式 说明
注册奖励 新用户获得 100 ATP
出售 Token 他人购买你的 Agent Token 时获得
⚠️

重要提示：ATP 仅为平台内部积分，无任何现实货币价值，不可兑换、提现或转让。

Token 机制

Token 是访问 Agent 的凭证。不同类型的 Token 有不同的权限：

类型 权限 说明
full 完整访问 浏览知识库 + 问答交互 + 查看详情
qa_public 公开问答 有次数限制的问答，适合试用
browse_public 仅浏览 只能查看公开内容，不能交互

Token 通常有有效期，过期后需要重新购买。

浏览市场

在 市场页面 可以看到所有已发布的 Agent。每个 Agent 卡片显示：

  • 名称和描述 - Agent 的基本信息
  • 类型标签 - Human / Character / Simulation
  • 价格 - 购买 Token 所需的 ATP
  • 在线状态 - Agent 是否可访问

购买 Token

找到感兴趣的 Agent 后：

  1. 点击 Agent 卡片上的「购买」按钮
  2. 确认价格和 Token 类型
  3. ATP 从余额中扣除
  4. 获得 Token，可在仪表盘查看

使用 Agent

购买 Token 后，在 仪表盘 的「我购买的」区域可以看到你的 Token。

  1. 点击「复制」获取完整 Token
  2. 点击「访问」跳转到 Agent 界面
  3. 在 Agent 界面中使用 Token 进行交互

发布 Agent

要发布自己的 Agent，你需要先部署 Cogmate 作为 Agent 后端。

部署完成后：

  1. 在 仪表盘 点击「发布 Agent」
  2. 输入你的 Cogmate API 地址
  3. 点击「检测」自动获取 Agent 信息
  4. 补充描述和定价
  5. 提交发布

管理 Token

发布 Agent 后，你需要生成 Token 供用户购买：

  1. 在 Cogmate 中生成 Token
  2. 在 CogNexus 仪表盘找到你的 Agent
  3. 点击「管理 Token」
  4. 添加生成的 Token
  5. 设置单价

用户购买后，Token 会标记为已售，你会收到相应的 ATP。
