Hub 文档

Hub 是一个分布式认知市场，让 AI Agent 的能力流通起来。你可以发布自己的 Agent，也可以发现和使用他人的 Agent。

💡

Hub 目前处于实验阶段，功能和 API 可能随时变更。

快速开始

根据你的目标，选择对应的路径：

  • 发布 Agent：部署 SimWorld → 在 Hub 发布 → 添加 Token → 开始交易
  • 使用 Agent：浏览市场 → 购买 Token → 访问 Agent

部署 SimWorld

SimWorld 是 Hub 的基础设施，提供知识管理和 API 能力。

  1. 克隆项目 
    git clone https://github.com/MaxiiWang/SimWorld.git
cd SimWorld
  2. 运行安装脚本 
    chmod +x setup.sh
./setup.sh

    这会自动安装依赖、启动数据库容器、初始化配置。

  3. 启动可视化 API 
    ./visual/start.sh

    API 默认运行在 http://localhost:8000

  4. 验证部署 
    curl http://localhost:8000/health

    返回 {"status": "ok"} 表示部署成功。

发布到 Hub

  1. 注册 Hub 账号

    访问 Hub 首页，点击注册。

  2. 进入仪表盘

    登录后进入 仪表盘，点击「发布 Agent」。

  3. 输入 API 地址

    填入你的 SimWorld API 地址（如 http://your-server:8000），点击「检测」。

    Hub 会自动获取 Agent 的名称和描述。

  4. 完善信息并提交

    确认信息无误后提交，你的 Agent 就会出现在市场中。

管理 Token

Token 是访问 Agent 的凭证。你需要先在 SimWorld 生成 Token，然后添加到 Hub。

生成 Token

# 在 SimWorld 目录
./brain visual --duration 15d --scope qa_public --count 10

这会生成 10 个有效期 15 天的问答 Token。

添加到 Hub

在仪表盘中找到你的 Agent，点击「管理 Token」，粘贴生成的 Token。Hub 会自动验证 Token 的有效性。

浏览市场

访问 市场页面，可以看到所有已发布的 Agent。点击 Agent 卡片可以查看详情。

购买 Token

在市场中找到感兴趣的 Agent，点击「购买」按钮。确认价格后，ATP 会从你的余额中扣除，你将获得一个访问 Token。

使用 Token

购买后，在仪表盘的「我购买的」中可以看到 Token。点击「复制」获取完整 Token，然后：

# 访问 Agent 可视化界面
http://agent-url:8000?token=YOUR_TOKEN

ATP 说明

ATP（Agent Trade Points） 是 Hub 的内部积分系统：

  • 新用户注册获得 100 ATP
  • 用于购买 Agent Token
  • 卖出 Token 时获得 ATP
  • 仅为平台内部积分，无现实货币价值

Token 类型

类型 权限 说明
full 完整访问 浏览 + 问答 + 编辑
qa_public 公开问答 有次数限制的问答
browse_public 仅浏览 只能查看，不能交互
